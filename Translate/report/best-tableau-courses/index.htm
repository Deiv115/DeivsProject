What is Tableau?

Tableau is a market-leading visual analytics platform that enables organizations to use data to solve problems. This business intelligence software makes it easier for people to explore and manage data so that they can discover and share business insights faster.

Whether you’re an analyst, data scientist, student, teacher, executive or business user, Tableau is a powerful, secure and flexible end-to-end analytics platform that helps you see and understand data so that you can make successful business forecasts and decisions.

Why are Tableau Skills Important?

Salesforce (Tableau) is one of the leaders in the 2022 Gartner® Magic Quadrant™ for Analytics and Business Intelligence Platforms.

Forrester recognizes Salesforce (Tableau) as a leader in The Forrester Wave™: Augmented BI Platforms, Q3 2021. According to Forrester, in 2020, 49% of data and analytics decision-makers at firms with an advanced level of insights-driven business maturity expect their organization to use augmented BI platforms, compared with only 31% at firms with a beginner level of maturity.

In 2020, two-thirds of data and analytics decision-makers reported that their organization was recruiting more talent with advanced data and analytics skills. Moreover, data and analytics decision-makers said their organization allocated as much as 45% for combined investments in reporting, dashboards, performance management, data exploration, and OLAP.

I’ve recently started learning data analytics from the IBM Data Analyst Professional Certificate. From there, I learned the importance of data, and how we can analyze them using Excel and IBM Cognos. Since the foundational concepts of data analytics are the same irrespective of the tool, it becomes easier for me to understand and use Tableau.

Besides that, I am a Guided Project Instructor and a Beta Tester at Coursera, having tested many courses before they’re officially launched. I have taken over 50 online courses in various subjects.

My experience as an online learner and teacher has given me some perspective on what to look for in an online course. I used my experience to evaluate each course in this list.

Course Ranking Methodology

I built this ranking following the now tried-and-tested methodology I used in previous rankings (you can find them all here). It involves a three-step process:

  1. Research: I started by leveraging Class Central’s database with 100K online courses and 200K+ reviews. Then, I made a preliminary selection of 150+ Tableau courses by rating, reviews, and bookmarks.
  2. Evaluate: I read through reviews on Class Central, Reddit, and course providers to understand what other learners thought about each course and combined it with my own experience as a learner.
  3. Select: Well-made courses were picked if they presented valuable and engaging content and they have to fit in a set of criteria and be ranked accordingly: comprehensive curriculum, release date, affordability, ratings and enrollments.

The end result is a unique selection of courses that combines a decade of Class Central data and my own experience as an online learner to try to get the best of both worlds. So far, I’ve spent more than 12 hours building this list, and I’ll continue to update it.

My first pick for the best Tableau online course is Tableau 2020 A-Z: Hands-On Tableau Training for Data Science on Udemy.

Next, you’ll learn how to work with data blending in Tableau, create table calculations, work with parameters, create dual axis charts and calculated fields, export results from Tableau into Powerpoint, Word and other software, work with time series data, and create data extracts in Tableau.

Finally, you’ll understand aggregation, granularity and level of detail, add filters, create data hierarchies, add actions to dashboards such as filters and highlighting, assign geographical roles to data elements, and advanced data preparation including latest updates in Tableau.

How You’ll Learn

3. Tableau for Data Scientists (LinkedIn Learning)

My third pick for the best Tableau online course is Tableau for Data Scientists on LinkedIn Learning.

In order to take this course, it’s expected that you already have some basic understanding of Tableau. Aimed towards intermediate data analysts, this course builds on your existing Tableau skills, and moves up to more advanced features that will help you level up your Tableau game. A one-month free trial is also available.

What You’ll Learn

In the first module, you’ll learn what green and blue fields mean. You’ll understand their difference, how they affect rows and columns, colors, filters and dates, and work on a challenge in these fields.

In the second module, you’ll learn about connecting to a source of data. How to connect to Excel sheets, how to clean Excel data with the data interpreter, how to connect to PDF files and extract tables of data, how to connect to a local database, work on a challenge on connecting to data.

In the third module, you’ll learn about combining data. You’ll understand combining data using relationships, what the different ways of joining data are, how to join tables in the same data connection, how to join tables using a cross-database join, how to append one data source to another using unions, and work on a challenge on combining data.

In the fourth module, you’ll understand when and how to create data extracts. You’ll learn what the pros and cons of using a data extract are, how to create a data extract, how to limit the data in a data extract, how to edit a data extract to include more data, and work on a challenge on working with extracts.

In the fifth module, you’ll learn about comparing measures. You’ll understand what measure names and measure values are, creating a combined axis chart, dual axis chart, bar chart and multiple measure crosstab, and enhance a crosstab using colors to create a highlight table.

You’ll also compare two measures using a scatter plot, enhance scatter plots using trend lines and animation, and work on a challenge on comparing measures.

In the sixth module, you’ll transform your data with calculations. You’ll learn how calculations work in Tableau, use calculations in a join, how table calculations work, give direction in table calculations, what level of detail (LOD) calculations are, calculate time durations, use IF calculations, and work on a challenge on calculations.

In the seventh module, you’ll learn about mapping your data. You’ll understand when to create a map, how to create an area map, how to create a symbol map, customize the look of your maps, create locations from coordinates, create lines to connect locations on a map, analyze distances using buffer calculations, and work on a challenge on mapping.

In the eighth module, you’ll explore analytics. You’ll learn how to use colors to highlight data, create reference lines, create reference bands, use motion to show changes over time, and work on a challenge on analytics.

In the ninth module, you’ll learn about using parameters for greater interaction. You’ll understand how parameters are different from filters, how to use parameters in calculations, create dynamic reference lines using parameters, use a parameter to select dimensions and measures, use a parameter to search free text fields, perform top N analysis, and work on a challenge on parameters.

Finally, in the tenth module, you’ll take a look at dashboard actions and design tips. You’ll learn how to use containers in a dashboard, create a tiled dashboard,  create a floating dashboard, create device-specific dashboards, use filter actions, use highlight actions, use URL actions, use set actions, and work on a challenge on dashboards.

How You’ll Learn

The course is broken down into 10 modules. Each module involves about 20 to 40 minutes of work. Concepts are taught through a combination of videos, practice exercises and quizzes.

8. Complete Tableau 2021 Training for Absolute Beginners (Udemy)

My eighth pick for the best Tableau online course is Complete Tableau 2021 Training for Absolute Beginners on Udemy.
